# dotvimi8 - Modern Vim 8+ configuration

Modern Vim8 configuration using the 'minpack' manager.

```
git clone git@github.com:templecloud/dotvim8.git ${HOME}/.vim
cd ${HOME}/.vim
./minpack-activate.sh
```
